- 使用 Vue Router 4
- 动态路由加载
- 路由守卫 (登录验证)
- 面包屑导航

### 状态管理

- 用户信息存储
- Token 管理
- 菜单权限管理

### UI 组件
- Ant Design Vue 4.x
- 自定义主题配置
- 响应式布局

### 数据可视化
- ECharts 图表
- 交易统计图表
- 数据概览卡片
